# Docs Index

현재 C 런타임 기준으로 먼저 볼 문서는 아래 네 개다.

- `ANC_RT_SETUP.md`
- `AGENT.md`
- `handoff/NEXT_SESSION_PROMPT.md`
- `daily/YYYY-MM-DD.md`

문서 성격은 다음과 같다.

- `ANC_RT_SETUP.md`: 운영 설정과 측정 결과
- `AGENT.md`: 코드 수정 시 지켜야 할 구현 원칙
- `handoff/NEXT_SESSION_PROMPT.md`: 세션 인수인계 메모
- `daily/YYYY-MM-DD.md`: 날짜별 작업 로그와 검증 메모

앞으로 코드, 서비스, 측정, 튜닝을 건드린 날에는 `docs/daily/` 아래에 같은 날짜 이름으로 작업 로그를 남긴다.

과거 제품 조사와 Python 프로토타입 자료는 `../archive/ANC_Bed_Speaker/` 아래에 있다.
